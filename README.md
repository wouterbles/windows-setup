# Windows setup 
Fresh Windows 10 setup script that installs multiple packages and command line utilities making use of Scoop and AppGet.

## Scoop packages
- Git
- NodeJS LTS
- OpenSSH

## AppGet packages
- 7 Zip
- Spotify
- Chrome
- VLC
- Google Backup & Sync
- WhatsApp
- Acrobat Reader
- Authy
- Bitwarden
- Calibre
- Deluge
- Logitech Options
- Visual Studio Code
